Lesson 3 Highlights

Lonnie McMillon

Highlight #1 Git

Git is a widely-used version control repository. It stores changes (commits) and makes comments that developers use to distinguish their changes. It uses a system of pushing (uploading) and pulling (downloading) files to allow seemless cooperation between developers.

Highlight #2 Short Status

Github and some repositories feature a special type of noting for commits. This noting is called short status as normal git commits are compact. This saves the developer time when reading over changes as it gives summaries and not long noted changes.

Highlight #3 SSH

SSH (Secure Shell) is a protocol used to link to remote servers securely. Github uses this kind of connection to make sure developers data and commits are secured as they work and push/pull their commits and files. This also establishes a secure form of sharing between developers.
